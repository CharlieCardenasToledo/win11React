¿Qué es Simulación?

"La simulación es el proceso de diseñar un modelo de un sistema real y llevar a término experiencias con él..." - Shannon

En palabras simples

Una copia virtual para experimentar sin riesgos.

Ejemplos

  • Videojuegos
  • Simuladores de vuelo
  • Pronóstico del clima
  • Crash tests

Limitaciones de la Simulación

Cuándo NO simular

  • Solución analítica disponible
  • Experimentación directa viable
  • Falta de datos
  • Recursos insuficientes

Desafíos comunes

  • Complejidad del modelo
  • Validación
  • Interpretación
  • Tiempo de desarrollo

Conceptos Básicos

  • Eventos aleatorios

  • Métricas
import random
def simular_llegadas(n_clientes, tiempo_llegada):
    tiempos = []
    t = 0
    for _ in range(n_clientes):
        t += random.expovariate(1 / tiempo_llegada)
        tiempos.append(t)
    return tiempos
